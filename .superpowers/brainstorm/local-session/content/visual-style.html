选一个适合"打字练习"的整体视觉风格

这里展示三种方向，重点看是否觉得"贵"，是否愿意在上面花时间练习。

A

新中式纸韵

灵感来自宣纸质感与留白美学，以温暖的深棕色为主调，配中文字体「霞鹜文楷」。卡片有轻微投影，按钮用深棕底白字。适合有文化底蕴的打字练习场景。

B

现代极简深色

深灰/炭黑背景，搭配暖琥珀色高亮。参考 Obsidian、Arc Browser 的暗色主题。阅读舒适、长时间练习不刺眼。卡片用半透明暗灰区分层级。

C

清爽冷白

纯白底 + 冷灰层次 + 靛蓝色强调。参考 Notion、Linear 的干净感。卡片用细边框分隔，无投影。整体偏理性、中性，不抢注意力。
