文章卡片的视觉样式

文章列表里每篇文章的展示方式，直接影响整体质感

A

纯白卡片 + 微投影 + 左侧装饰条

白色背景，0 2px 12px rgba(0,0,0,0.06) 投影，左侧一条 4px 宽的深棕色装饰条（标示文章类型或状态）。hover 时投影加深。标题粗体，预览文字两行截断，标签在底部用小胶囊。整体精致且有层次。

B

无投影卡片 + 细边框

白色背景，1px solid #E8E2DA 细边框分隔，hover 时边框变深棕。无投影，依赖边框和背景色差异创造层级。更偏扁平、干净。标题粗体，预览文字两行截断。

C

毛玻璃卡片

半透明白背景 + backdrop-filter: blur(12px)，卡片边缘有 1px 的半透明白色描边。投影更深。视觉上更有现代感和深度，但在 WPF 中实现需要自定义 Effect 或 Shader，有一定实现成本。

关键差异

维度 A. 微投影+装饰条 B. 细边框 C. 毛玻璃
层次感 ★★★★ 投影+装饰条双层营造 ★★☆☆ 依赖边框，较平 ★★★★★ 最有深度
WPF 实现 简单 — DropShadow + Border 最简单 — 纯 Border 复杂 — 需 Shader Effect
与纸韵风格匹配 高 — 装饰条可呼应深棕主色 中 — 偏扁平现代 低 — 偏未来感
