# WPF 视觉美化设计

> 日期：2026-07-03
> 范围：阶段七 WPF 前端视觉样式重构
> 目标：消除当前界面的"廉价感"，建立统一、精致、有文化底蕴的视觉语言

---

## 1. 背景与问题

阶段七已实现主窗口导航、文章库页和设置页的功能骨架，但视觉层面存在以下问题：

- 大面积米黄/暖棕色拼接，色彩廉价感明显
- 侧边栏、内容区、卡片之间缺乏明确的视觉层级
- 橙红色按钮过于刺眼，与背景不协调
- 侧边栏导航粗糙，缺少图标和精致的选中态
- 排版松散，文字间距不统一
- 卡片无投影，所有面板都是纯色背景+圆角，没有深度
- 搜索框简陋，原生控件与整体风格脱节

## 2. 设计决策

通过视觉伴侣对比，确认以下四项关键决策：

| 决策项 | 选择 | 理由 |
|--------|------|------|
| 整体风格 | 新中式纸韵 | 温暖深棕主调 + 留白美学，与打字练习的文化调性匹配 |
| 侧边栏 | 圆角色块 + 图标 | 选中项深棕底白字圆角色块，配 SVG 图标，现代精致 |
| 文章卡片 | 微投影 + 左侧装饰条 | 白色背景 + DropShadow + 4px 深棕装饰条，层次感好 |
| 控件风格 | 圆角胶囊按钮 + 内嵌搜索 | 与导航统一的圆角语言，柔和舒适 |

## 3. 色彩体系

| 角色 | 色值 | 用途 |
|------|------|------|
| 纸白背景 | `#FAF8F5` | 主内容区底色 |
| 侧边栏 | `#F3EFEB` | 与主内容区微妙区分 |
| 卡片白 | `#FFFFFF` | 卡片、输入框背景 |
| 主强调 | `#8B5E3C` | 按钮、选中态、装饰条 |
| 主强调 hover | `#7A5234` | 按钮 hover 态 |
| 次强调 | `#C4A882` | 标签胶囊、辅助装饰 |
| 正文 | `#2D2926` | 主要文字（对比度 > 7:1） |
| 次级文字 | `#7A7067` | 说明文字、时间戳 |
| 边框 | `#E8E2DA` | 输入框、分隔线 |
| 错误 | `#C75146` | 错误状态 |
| 成功 | `#5B8C5A` | 导入成功等正反馈 |

所有文字与背景组合均满足 WCAG AA 4.5:1 对比度要求。

## 4. 排版规范

| 层级 | 字号 | 字重 | 用途 |
|------|------|------|------|
| 页面标题 | 28px | Bold (700) | "文章库"、"设置" |
| 卡片标题 | 18px | SemiBold (600) | 区域标题 |
| 正文 | 14px | Regular (400) | 说明文字 |
| 辅助 | 12px | Regular (400) | 时间戳、标签 |
| 导航项 | 15px | SemiBold (600) | 侧边栏按钮 |

行高统一 1.5，说明文字行高 1.6 以提升可读性。

## 5. 组件设计

### 5.1 侧边栏导航

- 宽度 280px，背景 `#F3EFEB`
- 顶部应用标题"打字练习" 28px Bold
- 导航项：圆角 14px 色块，选中时深棕底 `#8B5E3C` + 白字，未选中时透明背景 + `#4C443C` 文字
- 每个导航项左侧 24x24 SVG 图标（书本/齿轮），1.5px 描边
- 导航项高度 48px，左右内边距 16px，项间距 8px
- 底部说明卡片：`#F8F3EA` 背景，圆角 18px，辅助说明文字

### 5.2 文章卡片

- 白色背景 `#FFFFFF`
- 投影：`0 2px 12px rgba(0,0,0,0.06)`
- 左侧 4px 宽深棕色装饰条 `#8B5E3C`
- 圆角 16px
- 内边距 20px
- hover 态：投影加深至 `0 4px 16px rgba(0,0,0,0.10)`
- 标题 18px Bold，预览文字 14px Regular 两行截断
- 标签胶囊：`#F0E6D6` 背景 + `#8B5E3C` 文字，圆角 999px，内边距 8px 4px

### 5.3 按钮控件

- 主按钮：深棕底 `#8B5E3C` + 白字，圆角 10px，hover `#7A5234`
- 次按钮：透明底 + `#8B5E3C` 描边 1px + `#8B5E3C` 文字，hover 浅棕底 `#F0E6D6`
- 高度 40px，左右内边距 18px
- 字号 14px SemiBold
- 禁用态：opacity 0.5

### 5.4 输入控件

- 搜索框：无外边框，底部 1px `#E8E2DA` 分隔线，聚焦时变 `#8B5E3C`
- 左侧搜索图标占位（SVG，20x20，灰色）
- 高度 40px，内边距 12px
- 下拉框：圆角 10px，1px `#E8E2DA` 边框，聚焦时 `#8B5E3C`

### 5.5 内容区容器

- 主内容区背景 `#FAF8F5`
- 顶部标题栏：白色卡片 `#FFFFFF` + 微投影，圆角 20px，内边距 28px 24px
- 内容容器：白色卡片 + 微投影，圆角 20px，内边距 24px

## 6. 图标方案

采用内嵌 SVG Path 方案，零依赖：

- 文章库图标：书本轮廓（开合书本造型）
- 设置图标：齿轮轮廓
- 搜索图标：放大镜
- 统一 1.5px 描边宽度，圆角线帽
- 尺寸：导航项 20x20，搜索框 18x18

## 7. 改动范围

### 改动文件

| 文件 | 改动内容 |
|------|---------|

| `TypingCore.Wpf/Views/SettingsView.xaml` | 卡片和控件统一风格 |

### 不改动

- Core 层完全不动
- ViewModel 层逻辑不变
- 测试不变（测试的是行为，不是外观）
- 不引入新 NuGet 包（SVG 图标用内嵌 Path 实现）

## 9. 已知局限

- 本次只做静态视觉样式，不涉及主题切换运行时逻辑（阶段十三）
- 不引入动画过渡（hover/选中态的动画过渡留待后续打磨）
- 不做暗色模式（阶段十三统一规划）
